hpi:custom-war

Full name:

org.jenkins-ci.tools:maven-hpi-plugin:2.7:custom-war

Description:

Builds a custom Jenkins war that includes all the additional plugins referenced in this POM.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: runtime.
  • Binds by default to the lifecycle phase: package.
  • Invokes the execution of the lifecycle phase compile prior to executing itself.
